# TRABALHO
trabalho da 2º unidade - Lara e Wendra




O aplicativo  será de controle de vendas e estoque de protudos

em estoque pode conter:
nome produto
id produto
quantidade
material

controle de vendas:
nome cliente
telefone
cpf
nome loja (se vender o produto em lojas)
quatidade produto
id produto

Este projeto tem como objetivo desenvolver um sistema web para ajudar no controle de vendas e do estoque de uma loja ou fábrica. A ideia é reunir as principais funções do negócio em um único lugar, facilitando a organização e o gerenciamento do dia a dia.
O sistema permitirá cadastrar produtos, clientes e fornecedores, registrar entradas e saídas de mercadorias, acompanhar a quantidade de produtos em estoque e controlar as vendas realizadas. Dessa forma, será mais fácil manter o estoque organizado, evitar erros e tornar a administração do negócio mais prática e eficiente.
